【このツールについて】
このツールは、Java の基本を 順番に体験しながら学べる 初心者向け教材です。
まずは「① Hello World」から進めてみてください。

【ランチャーの使い方】
画面には次のボタンがあります。
- はじめに … この説明を表示します
- ① Hello World … 最初のプログラム
- ② 変数とデータ型
- ③ 条件分岐
- ④ 繰り返し処理
- ⑤ メソッドの定義と呼び出し
- ⑥ クラスとオブジェクト
- ⑦ コンストラクタとオーバーロード
- ⑧ 配列と ArrayList
上から順番に進めると理解しやすくなります。

【コードを入力して実行できます】
各課題では、自分で入力したコードをその場で実行できます。
内部で JavaCompiler を使っており、実際にコンパイル・実行されます。

【実行結果は教育係に提出します】
課題ごとに実行した内容は、担当の教育係に提出して確認してもらいます。
理解度チェックやアドバイスに役立ちます。

【トラブルが起きたとき】
- exe が起動しない
- ボタンが反応しない
などがあれば、再起動やフォルダ移動で改善することがあります。
解決しない場合は制作者に相談してください。

【最後に】
このツールは、初めての Java を迷わず学べるように 作られています。
焦らず、一歩ずつ進めていきましょう
